# Y-Haplogroups

Bilingual interactive guide to 26 Y-DNA haplogroups and human paternal lineage.

[![Live Demo](https://img.shields.io/badge/demo-y--haplogroups.vercel.app-teal)](https://y-haplogroups.vercel.app)

## Overview

A bilingual (English / Chinese) single-page site exploring Y-chromosome haplogroups -- the genetic markers that trace paternal ancestry across tens of thousands of years. Features include:

- **Animated DNA Helix** -- canvas-rendered hero animation
- **Phylogenetic Tree** -- interactive tree canvas with hover tooltips showing branching relationships
- **Haplogroup Catalog** -- filterable cards for all 26 major haplogroups, color-coded by geographic region
- **Reference Table** -- complete data table with origins, distributions, and estimated ages

## Design & Tech

- **Stack:** Single HTML file, vanilla JavaScript, HTML5 Canvas
- **Aesthetic:** Dark natural-science theme with region-based color coding
- **Typography:** Bitter (headings) + Nunito Sans (body) + Fira Code (data)
- **Interactions:** Canvas tree with hover tooltips, region filter buttons
- **Deployment:** Vercel
